What is happening inside the box?

The top row holds laser diodes aimed at the lower sensing area. Each beam lands on a matching photoresistor. When the beam stays uninterrupted, the sensor reads a steady light value. When a hand crosses the beam, the light level drops. The Arduino reads that change on an analog input, compares it to a threshold, then converts that event into a MIDI note. Logic Pro receives the note and plays whatever instrument patch you assign. Tiny science fair. Big synth energy.

Signal chain

The instrument works like a translation machine between light and sound. Every blocked beam is treated like a digital performance gesture, but the sensing starts in the analog world. That means your thresholds, resistor values, alignment, and code logic all matter a lot.

Design notes for the classroom version

This model is intentionally simplified so it communicates the build clearly. The real prototype has handmade spacing, visible jumper wires, cardboard structure, and the charmingly scrappy geometry that makes a classroom invention feel alive. For a more reliable version, the biggest upgrades would be laser alignment brackets, opaque dividers between beams, fixed mounts for the photoresistors, and a front panel that hides stray ambient light. That last part matters because photoresistors can get moody when room lighting starts doing side quests.
